## Getting started

```bash
# clone the project
git clone https://github.com/FGDevelop/handlestyle-api.git
```
please create database handlestyle
```bash
# create database FG-YouthUnion

# install composer laravel
composer install

# setup project
php artisan FGDev:setup
```

## Build

```bash
# turn on server
php artisan serve
```
